Features included

  • Want to rename your todo title? Just edit the title and press enter or click refresh icon to rename it
  • Adding the task is as easy as typing the text. Just add task description, then press enter or click on the icon and it will save it in the list. Task with no description are not allowed
  • Had a little typo while adding the task? No worries, just click on the item you want to edit. Once you are done just hit enter
  • Want to prioritize your to-do? Use space below three-dots icon or simply hold the task and drop it wherever you want
  • Done with the task and want to mark it as a complete? Simply click on checkbox to mark it as complete. It will strike of the task from the list
  • Want to remove the clutter from your todo? Simply click on clear completed task after marking all the task to complete
  • Delete icon will appear when you click on one of the tasks which can be used to delete single task from the list
  • If completed tasks are edited it will be transformed to incomplete tasks
  • Want your todo list back same as you left? Don't worry we will save every action on your browser because we know its important for you :)
